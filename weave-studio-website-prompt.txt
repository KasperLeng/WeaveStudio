# Weave Studio — Website Build Prompt

---

## Project Overview

Build the full marketing website for **Weave Studio**, a web design studio that creates
customized websites for clients. The site is a single scrollable page that functions as
both a portfolio showcase and a client acquisition tool.

The brand voice is: confident, minimal, professional, creative. Never salesy or corporate.

---

## Design System

Reference file: weave-studio-design-system.txt

Quick summary:
- Font: DM Sans (Google Fonts) — weights 400 and 500 only
- Page background: #F4F6F8
- Surface (cards, nav): #FFFFFF
- Primary action: #185FA5 (Anchor Blue)
- Hover / highlights: #378ADD (Clarity Blue)
- Input borders: #B5D4F4 (Mist Blue)
- Tinted backgrounds / badges: #E6F1FB (Frost Blue)
- Dark headings / overlays: #042C53 (Midnight)
- Secondary text: #888780 (Slate)
- Spacing: multiples of 8px
- Card border radius: 12px
- Button border radius: 6px
- Max content width: 1120px, centered, 40px page margins

---

## Tech Stack

- Single-file HTML (index.html) with embedded CSS and vanilla JS
- No frameworks or build tools required
- Google Fonts import: DM Sans 400 + 500
- Smooth scroll behavior on all anchor links
- All sections on one page with id anchors for nav links

## Behavior Notes

- All nav links scroll smoothly to their section using scroll-behavior: smooth on html element
- Active nav link detection via IntersectionObserver on each section (threshold 0.4)
- Filter pills on the Work section use JS to toggle a "hidden" class (opacity + pointer-events)
  with a 200ms CSS transition on opacity
- Form submit is handled client-side only (no backend needed) — just show the success message
- All external links (portfolio cards) open in a new tab with rel="noopener noreferrer"
- Page should be fully responsive:
  - Below 768px: all two-column layouts stack to single column
  - Nav collapses to a hamburger menu on mobile (simple toggle)
  - Font sizes scale down ~15% on mobile
